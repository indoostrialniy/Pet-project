
## Pet-Project version 0.1.1
**Small first person shooter powered by my self-maded engine using C++/OpenGL + GLFW, GLEW, GLM, SFML, STB, nameof.**
<br>**Asset export via Blender 3D-addon** 
[CEAE v0.3.10]( https://github.com/indoostrialniy/CEAE)

### Goals
**Demonstration of programming skills, scripting, working with 3D-graphics, mathematics, collisions, skeletal animations, shaders, illumination, materials, resource management, CMake, Git etc.**

### Launch
**Download latest release, unpack, run executable.**

## Демо-проект версия 0.1.1
*Небольшой шутер от первого лица на самописном движке, созданном на C++/OpenGL с помощью стандартных библиотек + GLFW, GLEW, GLM, SFML, STB, nameof.*
*Ассеты экспортируются с помощью аддона для Blender 3D.*
*Предоставляется "как есть" в демонстрационных целях.*

## Цели проекта
*Демонстрация навыков программирования, скриптования, работы с 3Д-графикой, математикой, скелетными анимациями, коллизиями, шейдерами, освещением, материалами, менеджментом ресурсов, CMake, Git и т.п.*

## Запуск
*Скачать последний релиз, распаковать, запустить исполнительный файл.*

### Libraries versions:
	GLFW 3.4
	SFML 2.6.1
	GLM 0.9.9.8
	GLEW 2.2.0
